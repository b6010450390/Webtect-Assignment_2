จุดประสงค์เพื่อการฝึกการใช้เฟรมเวค

CSS: Cascading Style Sheet

     CSSย่อมาจาก Cascading Style Sheet มักเรียกโดยย่อ ว่า "สไตล์ชีต" คือ ส่วนของการจัดรูปแบบการแสดงผลเอกสาร HTML โดยที่ CSS ทำการกำหนดกฏเกณฑ์ในการระบุรูปแบบ (หรือ "Style") ของเนื้อหาในเอกสาร ได้แก่ สีของข้อความ รูป สีพื้นหลัง ประเภทตัวอักษร และการจัดวางข้อความ

CSS Framework

     CSS Framework ก็คือ การจัดเตรียม โครงสร้างหน้าเว็บ ขั้นพื้นฐานเอาไว้ ให้เราเลือกปรับแต่งขนาดได้เองตามใจชอบ

ประโยชน์ของ CSS

     ช่วยให้เนื้อหาภายในเอกสาร HTML มีความเข้าใจได้ง่ายขึ้นและในการแก้ไขเอกสารก็สามารถทำได้ง่ายกว่าเดิม การใช้ CSS จะช่วยลดการใช้ภาษา HTML ลงได้ในระดับหนึ่ง และแยกระหว่างเนื้อหากับรูปแบบในการแสดงผลได้อย่างชัดเจน

ทุกคนคงรู้จักกับ css กันแล้วนะครับ แต่ผมก็เชื่อว่าหลายๆคนอาจจะเพิ่งเคยได้ยินชื่อนี้ “CSS Specificity”

     CSS Specificity คือ Css Specificity คือค่าบางอย่างที่จะจะบอก browser ว่าให้เลือกใช้ Style ตัวไหนมาแสดงผลหน้าเว็บ แน่นอนว่าจะต้องมีการคำนวนค่าออกมาเพื่อให้ browser รู้ได้นั่นเอง

การคำนวณค่า CSS Specificity

     เราสามารถสร้าง style ให้กับ element ได้หลายแบบ ที่นิยมเขียนกันมากที่สุดก็จะเป็นการเขียนแบบใช้ selector ซึ่งการเขียนแบบนี้ก็มีหลายแบบอีก เช่น

- Type selectors คือการใช้ element เป็น selector เช่น h1, input, image
- Class selectors อันนี้รู้จักกันดี เช่น .container, .wrapper
- ID selectors ใช้ id เป็น selector เช่น #name, #profile
- Attributes selectors เช่น [type=“radio”]
